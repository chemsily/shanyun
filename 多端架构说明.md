# 衫云智管 · 多端架构

> 一套核心代码，运行在 Web / PWA / 小程序 / App

## 架构设计

```
┌─────────────────────────────────────────────────────┐
│                  业务代码层                          │
│  app.js / features.js / ai-engine.js                 │
│  (纯 JS，无平台依赖)                                │
└─────────────────────────────────────────────────────┘
                        ↓ 调用
┌─────────────────────────────────────────────────────┐
│              PlatformAdapter 适配层                  │
│  platform-adapter.js (本文档说明)                   │
└─────────────────────────────────────────────────────┘
                        ↓ 分发到
┌──────────┬──────────┬──────────┬──────────┬──────────┐
│   Web    │   PWA    │  WeApp   │  Alipay  │   App    │
│ localSt. │IndexedDB │wx.* API  │my.* API  │ uni.*    │
│ fetch()  │SW缓存    │wx.request│my.request│ uni.*    │
└──────────┴──────────┴──────────┴──────────┴──────────┘
```

## 平台能力抽象

`PlatformAdapter` 提供了统一的 API，业务代码无需关心平台差异：

### 存储
```js
// 自动适配：localStorage / IndexedDB / wx.setStorageSync / my.setStorageSync
await PlatformAdapter.storage().set('key', value);
const v = await PlatformAdapter.storage().get('key');
```

### 网络请求
```js
// 自动适配：fetch / wx.request / my.request
const data = await PlatformAdapter.request({
  url: '/api/products',
  method: 'GET'
});
```

### UI 交互
```js
// Toast、模态框、导航、图片预览 - 全平台统一 API
PlatformAdapter.toast('保存成功', 'success');
PlatformAdapter.navigate('/pages/products/list');
await PlatformAdapter.modal({ title: '提示', content: '确认删除？' });
PlatformAdapter.preview(['url1', 'url2'], 'url1');
```

### 媒体（拍照/相册）
```js
// 自动适配：input[type=file] / wx.chooseImage / my.chooseImage
const images = await PlatformAdapter.chooseImage({ count: 1, sourceType: ['camera'] });
const compressed = await PlatformAdapter.compressImage(images[0].tempFilePath, 0.75);
```

### 平台检测
```js
PlatformAdapter.platform         // 'web' | 'pwa' | 'weapp' | 'alipay' | 'uniapp-h5' | 'uniapp-app'
PlatformAdapter.isWeb            // true
PlatformAdapter.isMiniProgram    // false
PlatformAdapter.isApp            // false
```

## 各端部署

### 1. Web / PWA（一键）

```bash
./deploy.sh web      # 部署 Web 版
./deploy.sh pwa      # 部署 PWA 版（自动启用 Service Worker）
```

部署到 GitHub Pages 后，用户可在 Chrome 地址栏看到"安装"按钮，可安装为 App。

### 2. 微信小程序

```bash
./deploy.sh weapp
```

步骤：
1. 安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 导入项目目录 `./weapp`（需将 `app.json` 复制到 `weapp` 目录）
3. 在 `project.config.json` 中填入 AppID
4. 点击「上传」按钮提交到微信后台审核

### 3. uni-app App / H5

```bash
npm install -g @vue/cli @vue/cli-init
cd multi-platform
npm install
npm run build:app        # 打包 App
npm run build:mp-weixin  # 打包微信小程序
npm run build:h5         # 打包 H5
```

## 核心原则

1. **业务代码零平台依赖** - `ai-engine.js` 等核心模块不调用 `wx.*` `my.*` 等平台 API，全部通过 `PlatformAdapter` 访问
2. **Adapter 自动检测平台** - 运行时根据全局对象（`wx`/`my`/`uni`）自动选择实现
3. **渐进式增强** - Web 版本无平台特性，PWA/App/小程序逐步增强
4. **一次开发，多端运行** - 核心业务代码 100% 复用，UI 层通过 uni-app / Taro 等工具转换
